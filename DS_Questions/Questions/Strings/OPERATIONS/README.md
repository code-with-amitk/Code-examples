## Traversal
```cpp
#include<sstream>
int main(){

  string str = "Fail early fail often";
  
  for (auto i=str.begin(); i!=str.end(); i++)       //Forward iterator
    cout<<*i;
  
  for (auto i=str.rbegin(); i!=str.rend(); i++)    //Reverse iterator     
    cout<<*i;
}

/*O/P:
Fail early fail often
netfo liaf ylrae liaF */
```
